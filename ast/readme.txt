[AST 맛보기]

-------------------------------------------------------------------------

1. ast 추출 (추출할 파일의 Line of Code는 500중 이상의 오픈소스 활용):
python3 ./generate_ast.py ./1stsource.c >> 1AST.json

이렇게 하면 1AST.json 파일이 생성된다

-------------------------------------------------------------------------

2. 파일 분석 조건 (astparser.c):
-> 함수 개수 추출
-> 함수 리턴타입 추출
-> 함수 이름 추출
-> 함수 파라미터 타입, 변수명 추출
-> 함수 if 조건 개수 추출

-------------------------------------------------------------------------

2-1. 분석에 쓰일 코드 컴파일:

gcc -o 프로그램명 .\astparser.c cJSON.o


-------------------------------------------------------------------------

2-2. 프로그램 실행:
./parser 1AST.json

-------------------------------------------------------------------------
